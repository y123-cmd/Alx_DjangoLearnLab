# LibraryProject

This is a Django project named LibraryProject.

## Installation

1. Clone this repository.
2. Install Django using pip: `pip install django`.
3. Start the development server: `python manage.py runserver`.

## Project Structure

- `LibraryProject/`: Main project directory.
  - `LibraryProject/`: Contains settings and configurations.
    - `settings.py`: Configuration settings for the Django project.
  - `urls.py`: URL declarations for the project.
  - `manage.py`: Command-line utility to interact with the Django project.

## Usage

1. Navigate to the project directory.
2. Start the development server.
3. Open a web browser and go to `http://127.0.0.1:8000/`.
